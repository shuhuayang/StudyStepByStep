## k线
* k线的意义
```bash
k线是一门投资哲学，它反映了当天多空双方交战胜负的结果，记录着每天人们成败得失的全部过程，
它承载着各种类型资金买入卖出的踪迹，是股市运行中最基本的表现符号。
k线涨跌是股市行情走势的表象，而形成阴阳k线的根本原因是交易，是不同人群为了各自的目的，对股票看好或者看空，并做出买入卖出的行为。
买卖背后隐藏的是人心的变化，人性心理波动才是股市k线涨跌的本质。

k线图展现的是过去发生的事，虽然从理论上讲，市场的未来走势不可预测；
但我们最大限度所能做的事，就是分析识别k线波动与股价涨跌背后的规律，这可以提高我们判断未来股价波动的准确性，
```
* k线定义
```bash
一根k线是由4大核心价格元素组成的：分别是开盘价、收盘价、最高价和最低价。其中收盘价最关键的，它代表多空双方经过一天的激烈交战后，当天市场对股票的最终定价。
每根k线所传递的三个重要信号：
1. k线的颜色，传递阴阳涨跌的信号；
当天收盘价高于开盘价，表示上涨，这根k线被称为阳线，用红色表示；
当天收盘价低于开盘价，表示下跌，这根k线被称为阴线，用绿色表示；
参考牛顿的力学定律，当股票在没有受到外力影响时，价格仍将按原有方向与速度惯性运行。
也就是，今天如果收阳线，那就预示着明天k线初期，有惯性上冲需求；同理，如果今天k线收阴线，那就预示着明天k线初期，有继续下跌惯性。
2. k线实体大小，传递内在动力信号；
根据当天开盘价和收盘价之间的差距，可以判断多空双方内在动力的强弱。
当他们的力量差距越大，那么那根阳线或阴线的实体越大；比如你看到一根实体很大的阳线，你会想到什么呢？
它其实代表的是，当天多方做多的气氛浓厚，上涨力度强劲。而中阳线、小阳线相应表明做多气氛和上涨力度就依次在递减。
同样的道理，大阴线代表的是，空头做空气氛浓厚；而中阴线或小阴线，就表明做空意愿没有那么强烈。
3. k线影线长短，传递多空买卖的强弱信号；
k线柱子实体顶部的价格和当日最高价的连线，就是上影线，它代表多方大举进攻受阻，盘中先涨后跌；
k线柱子实体底部的价格和当日最低价的连线，就是下影线，它代表空方不断打压股价后多头反攻，盘中先跌后涨；
简单来说，上下影线记录着当天多空双方争夺地盘时，曾经到达过的最远阵地，但最终受到另一方的强力狙击，被折回来以后，留下来的痕迹。
而影线的长短所代表的意义也不同；当上影线越长，就表明空头反扑，阻挠多方上涨的力度越大；
那下影线越长，就表明多方反攻，阻挠空头下跌的力度越大。

根据不同的时间周期数据可以组成不同的k线图，比如5分钟线、日k线、月k线等；通过不同的k线周期组成分析，可以得出不同阶段未来股价的运行规律。
根据k线图所形成不同的价格区间，和不同形态走势，以及多空双方交战胜负结果，分析出它们的运行规律，以便帮助我们做出正确的买卖决策。
```
* k线动力分析
```bash
k线动力从字面上理解就是推动股价涨跌背后的动力之源。
参考牛顿力学原理，当列车启动前进，必须要有很强的外力推动着它，列车才能向前行驶；同样，想让列车从运动中停下来，也需要有外力作用。
如果把k线涨跌运行拿来同火车前进停止作比较，我们就可以发现k线的涨跌背后同样存在着一股内在的动力。
而影响股价的涨跌、股价的强弱、股价的运行方向就是我们研究股价k线涨跌动力的核心。
```
